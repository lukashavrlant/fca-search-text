Obsah CD

Obsah CD je rozdělen do tří adresářů V adresáři bin jsou zdrojové soubory připravené ke běhu. V archivu fca-search.zip jsou zdrojové soubory jádra vyhledávače. Vyhledávač je napsaný v Pythonu 3, takže ke spuštění je nutný interpret Pytyhonu. V archivu fca-search-web.zip jsou zdrojové soubory webového rozhraní. V adresáři doc je textová verze diplomky a zdrojové soubory v LaTeXu a obrázky. V adresáři src jsou zdrojové soubory obou aplikací. Protože aplikace nemají žádnou binární formu, je obsah téměř stejný jako v adresáři bin (nenachází se zde aplikace pdftotext apod.).

Online demo vyhledávače

Základní vyhledávání

Současná verze je nainstalována na adrese phoebe.inf.upol.cz/claseek. Je tam vytvořeno celkem pět testovacích indexů, čtyři české a jeden anglický. Vlajky přepínají mezi českým a anglickým seznamem indexů. Popis indexů spolu s ukázkovými dotazy:

  • inf je index webu katedry informatiky. Můžete zde klást dotazy jako fuzzy, práce, zkoušky nebo předměty. Výsledky jsou všelijaké, viz text diplomové práce.
  • jpwi je index webu jak psát web.cz, který obsahuje spoustu článků obecně o webu, HTML, CSS, Javascriptu apod. Můžete zde klást dotazy jako hosting php, použitelnost webů, formuláře nebo javascript cyklus.
  • jakpodnikat je index webu jak podnikat.cz, který obsahuje články o podnikání, daních apod. Můžete zde klást dotazy typu příjmy, slevy na dani, adsense dph nebo
  • matweb je index webu matweb.cz, který obsahuje články o středoškolské matematice. Můžete zde klást dotazy typu množiny, interval, limita funkce nebo bod.
  • articles je index anglických článků zaměstnanců KI, tedy především články se zaměřením na fuzzy, FCA a faktorová analýza. Můžete zde klást dotazy typu fuzzy logic, hedges, formal concept analysis nebo decomposition matrices.

Vyhledávač má implementovanou i triviální kontrolu překlepů, viz například dotaz ourtata.

Ukázka dotazovacího API

Vyhledávač poskytuje primitivní dotazovací API. Umí vracet informace o indexu a o jednotlivých dokumentech. API je popsáno v dokumentaci, takže jen několik málo příkladů:

  • seznam všech zaindexovaných URL v indexu inf
  • počet všech zaindexovaných dokumentů
  • seznam všech URL včetně jejich ID, které obsahují string jan
  • Informace o dokumentu s ID 81
  • Název stránky s ID 81

Ukázka zasílání vlastních dat vyhledávači

Vyhledávač dokáže přijmout přes HTTP POST sadu externích dokumentů, vytvořit dočasný index a provést nad daty FCA analýzu. Tohoto lze různě využít. SearchSleuth například zasílal dotaz, který mu uživatel položil, webovému vyhledávači, ten mu vrátil například 100 výsledků, SearchSleuth je zpracoval jako 100 různých dokumentů a nabídl úpravy dotazu. Pomocí tohoto API, které CLaSeek má, toto chování můžeme naprogramovat také.

Na adrese skola.havrlant.net/diplomka/api se nachází triviální aplikace, která toto API využívá. Jako první vložíme do textového pole dotaz. Aplikce odešle data do Google API, které vrátí 50 výsledků. Z těchto výsledků seskládáme dokumenty, zašleme je vyhledávači na adrese phoebe/claseek, ten provede FCA analýzu a aplikace zobrazí FCA část analýzy. Samotné dokumenty už nezobrazuje, je to čistě ukázka API. Příklady volání:

  • fuzzy logic
  • hollywood
  • limit of a function
  • palacky university

Aplikace pracuje pouze s anglickým Googlem, výsledky pro jiný jazyk mohou být všelijaké. Aplikace navíc používá neoficiální Google API, které nefunguje zrovna moc dobře. Občas nevrátí žádné výsledky, občas vrátí úplně jiné výsledky, než by to vrátit mělo, takže je to taková ruská ruleta.

Jak nainstalovat CLaSeek na vlastní server

Popis instalace a použití je popsán na Githubu, kde jsou také hostované zdrojové soubory. Na stránce github/fca-search/wiki/instalace je popsána instalace jádra vyhledávače. Instalace z CD je jednodušší, stačí překopírovat soubory a správná oprávnění adresářů už by měla být nastavena. Stejně tak se v adresáři /files/bin/ nachází aplikace pdftotext, ale pouze pro Mac OS X. Dokumentace pak je na stránce github/fca-search/wiki/dokumentace.

Popis instalace webového rozhraní se nachází na adrese github/fca-search-web/wiki/instalace a dokumentace na stránce github/fca-search-web/wiki/dokumentace. Webové rozhraní je naprogramováno v PHP.
